# Fitee
>**맞춤형 PT관리를 위한 핏한 선택!**<br>
>Fitee는 PT(퍼스널 트레이닝)와 비슷한 발음에서 착안하여, 트레이너와 회원을 FIT하게 맞춰주는 솔루션을 제공합니다. <br>
>클라이언트 관리부터 대화 요약까지! Fitee와 함께 스마트한 PT 관리의 시작을 경험하세요!
