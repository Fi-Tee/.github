# Introduction
> **맞춤형 PT관리를 위한 핏한 선택!**  
> Fitee는 PT(퍼스널 트레이닝)와 비슷한 발음에서 착안하여, 트레이너와 회원을 FIT하게 맞춰주는 솔루션을 제공합니다.
